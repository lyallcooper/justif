# Demo fonts

The demo serves its fonts from this directory so it has no runtime dependency
on a third-party font host.

- Junicode and Amstelvar are subsetted WOFF2 builds of the variable TTF source
  files kept alongside them.
- Alegreya, Courier Prime, EB Garamond, IBM Plex Mono, IM Fell English, Roboto
  Flex, and Vollkorn are the Latin WOFF2 subsets distributed by Google Fonts.
- Latin Modern is built from the OpenType files distributed by CTAN. It is the
  Unicode outline continuation of Computer Modern used by TeX.

The corresponding font licenses are included in this directory. Font URLs are
cached as immutable by Cloudflare Pages, so rename a file when replacing it.
